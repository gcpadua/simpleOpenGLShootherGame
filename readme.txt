
-----------------------------------
INSTRUÇÕES PARA EXECUTAR O MAKEFILE
-----------------------------------

1- Abra um terminal no caminho onde se encontra o arquivo MAKEFILE
2- O Passo 1 pode ser realizado no VsCode clicando com o botão direito em cima do arquivo MAKEFILE e clicar na opção "Abrir um terminal integrado".
3- Para rodar o comando "Compile": make compile
4- Para rodar o comando "Execute": make execute
5- Para rodar o comando "Clean": make clean
